# Bad Apple Spigot Plugin

一个在Minecraft服务器中播放Bad Apple视频的Spigot插件。

## 功能特性

- 🎵 播放Bad Apple音乐（通过资源包中的唱片）
- 🎬 在游戏世界中使用黑白混凝土方块播放Bad Apple视频
- ⏱️ 内置冷却系统，防止重复播放（3分55秒冷却时间）
- 🔧 可配置的视频墙位置和朝向
- 📦 预加载视频帧数据到内存，确保流畅播放

## 安装方法

1. 将生成的 `spigot-plugin-bad-apple-0.0.1-alpha2.jar` 放入服务器的 `plugins` 文件夹
2. 重启服务器或使用 `/reload` 命令
3. 编辑生成的配置文件 `plugins/spigot-plugin-bad-apple/config.yml`

## 配置文件

```yaml
# Bad Apple Plugin Configuration

# 视频播放墙体配置
video_wall:
  # 墙体左下角方块的坐标
  position:
    x: 0
    y: 64
    z: 0
  # 墙体朝向 (NORTH, SOUTH, EAST, WEST)
  direction: NORTH
  
# 播放设置
playback:
  # 是否启用视频播放功能
  enabled: true
  # 播放冷却时间（秒）
  cooldown: 235  # 3分55秒
```

### 配置说明

- **position**: 视频墙左下角方块的坐标
- **direction**: 墙体朝向，决定视频的显示方向
  - `NORTH`: 面向北方
  - `SOUTH`: 面向南方
  - `EAST`: 面向东方
  - `WEST`: 面向西方
- **enabled**: 是否启用视频播放功能
- **cooldown**: 播放冷却时间（秒）

## 使用方法

### 命令

- `/play_bad_apple` - 开始播放Bad Apple视频和音乐

### 权限

目前插件没有设置特殊权限要求，所有玩家都可以使用。

## 技术细节

### 视频规格

- **分辨率**: 384×216 像素
- **帧率**: 5 FPS（原15fps，每3帧取1帧）
- **更新频率**: 每4游戏刻更新一次（20tick/s ÷ 5fps = 4tick/frame）
- **总时长**: 3分55秒
- **墙体尺寸**: 384×216 方块

### 方块材质

- **白色像素**: 白色混凝土 (WHITE_CONCRETE)
- **黑色像素**: 黑色混凝土 (BLACK_CONCRETE)

### 数据格式

插件使用自定义的 `.bin` 文件格式存储视频帧数据：

1. **文件头** (8字节):
   - 前4字节: 宽度 (大端序)
   - 后4字节: 高度 (大端序)

2. **像素数据**:
   - 每个字节代表一个像素
   - 0 = 黑色像素
   - 1 = 白色像素
   - 按行排列（从左到右，从上到下）

## 注意事项

1. **服务器性能**: 视频播放会在短时间内更新大量方块，可能对服务器性能产生影响
2. **区域保护**: 确保视频墙区域没有被其他插件保护，否则可能无法正常更新方块
3. **世界备份**: 建议在专门的区域播放，避免覆盖重要建筑
4. **资源包**: 需要配合包含Bad Apple音乐的资源包使用以获得完整体验
